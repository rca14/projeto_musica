# Como executar

- Rodando o servidor: 
  - Entrar na pasta pm-server
  - npm install
  - npm start
- Front-end:
  - na raiz do projeto, abrir o arquivo index.html
  - Caso o passo anterior não funcione, executar: 
    - http-server -c-1
    - abrir link exibido no terminal
